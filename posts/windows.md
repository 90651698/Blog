# WINDOWS

## 触摸板操作

在 WINDOWS 设置中可以快速学习。

- **右键选中**：两根手指点击触摸板。
- **多选**：双击再拖动。
- **查看消息**：四指点击。
- **查看桌面**：三/四指下滑。
- **多任务视图**：三/四指上滑。
- **切换任务**：三指左/右滑。
- **search**：三指点击。

## 键盘操作

- **X1C 背光**：`fn + blank`。
- **多选**：`shift + 方向键`。
- **选中单词**：在单词开头，`shift + ctrl + ->`。
- **关闭软件窗**：`alt + F4`。
- **关闭页面**：`ctrl + w`。
- **锁屏**：`win + L`。

## 输入法

- **生僻字搜索**：使用自带输入法，输入 `u`，再输入 `又又又`，可查到 `叒`。
