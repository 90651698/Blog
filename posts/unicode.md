# UNICODE

- [UNICODE](#unicode)
  - [Box Drawing](#box-drawing)

最常用的是UTF-8，是一种针对Unicode的可变长度字符编码。

## Box Drawing

[参考](https://www.fuhaoku.net/block/Box_Drawing)

常用：

- `─`：U+2500
- `│`：U+2502
- `└`：U+2514
- `├`：U+251C

例：

```tex
Dataset
├──folder1
├──folder2
│  ├──item2.1
│  └──item2.2
└──folder3
   └──item3.1
```
