# UNICODE

- [UNICODE](#unicode)

最常用的是UTF-8，是一种针对UNICODE的可变长度字符编码。

<details>
<summary><b>BOX DRAWING</b></summary>

[[ref]](https://www.fuhaoku.net/block/Box_Drawing)

常用：

- `─`：U+2500
- `│`：U+2502
- `└`：U+2514
- `├`：U+251C

例：

```tex
Dataset
├──folder1
├──folder2
│  ├──item2.1
│  └──item2.2
└──folder3
   └──item3.1
```

</details>
